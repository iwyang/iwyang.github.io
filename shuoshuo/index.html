夸克网盘：利用“发送到手机”协议绕过限速。 操作步骤：

  • 定位文件：在电脑端网页或客户端登录夸克，找到需要下载的大文件。
  • 右键操作：选中文件后点击右键，选择菜单中的 “发送到手机”。
  • 进入传输记录：点击页面右上角的 “查看传输记录”（或类似的时钟/传输图标）。
  • 本地下载：在传输记录列表中找到该文件，点击上方的 “下载到本地” 按钮。

提示：如果发现速度没有提升，尝试切换一下代理 IP（如切换至海外节点），有时会触发不同的传输分配协议。

# 1. 进入下载目录
mkdir -p ~/Downloads
cd ~/Downloads

# 2. 下载最新的 2026.03 安装包
wget https://github.com/iwyang/backup/releases/download/koreader-v2026.03/koreader_2026.03-1_amd64.deb

# 3. 赋予读取权限 (习惯性动作，防止 sudo 模式下权限受阻)

# 4. 执行升级安装
# apt 会自动识别这是新版本并进行替换
sudo apt install ./koreader_2026.03-1_amd64.deb -y

# 5. (可选) 清理安装包以节省空间
rm ./koreader_2026.03-1_amd64.deb
